How it works
Чеки → AI → Solana → ₸
Четыре шага от фотографии чека до денег на кошельке
1
Сканируй чек
Пользователь фотографирует чек в Telegram Mini App. Поддержка RU, KZ, EN.
→
2
AI обрабатывает
Claude AI извлекает магазин, товары, цены, категорию. SHA256 хеш записывается на Solana.
→
3
Компании покупают
AI автономно подбирает аудиторию, назначает цену. Смарт-контракт выполняет оплату.
→
4
Деньги пользователю
~50 ₸ за чек. Автоматический вывод SOL на кошелёк. Полный контроль над данными.
Product
5 модулей, полностью рабочий продукт
Задеплоен и работает на app.aiqalam.kz
Telegram Mini AppB2C
Сканирование чеков через камеру, кошелёк с балансом SOL/₸, профиль пользователя, реферальная программа. 3 языка: RU/KZ/EN.
Company DashboardB2B
Интерактивная карта продаж по Казахстану, real-time аналитика, Zebra BI отчёты, спредшит каталог товаров, PDF экспорт.
Открыть Dashboard →
Admin PanelOPS
Модерация чеков (авто + ручная), управление выводами, аудит-лог всех действий, воронки пользователей, KPI дашборд.
Открыть Admin →
Smart ContractSOLANA
Anchor framework (Rust). Хранение хешей данных, автоматические платежи пользователям, верификация транзакций на Solana Devnet.
Backend API35+ API
Node.js + Express. AI-матчинг аудитории (Claude), OCR чеков, авто-модерация (дубликаты, аномалии), Telegram нотификации.
AI EngineCLAUDE
Claude Sonnet 4 для OCR чеков (фото → JSON) и автономного матчинга аудитории. AI сам решает кому продать данные и по какой цене.
